## Problem Statement

A full stack web app built with Node and Express that allows users to shorten long URLs (like bit.ly).

## How to run app

1. run `npm install`

2. run `node server.js`

3. Always delete cookies before running app again


## Dependencies

- Node.js
- Express
- EJS
- bcrypt
- body-parser
- cookie-session
